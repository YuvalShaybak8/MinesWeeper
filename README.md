# MinesWeeper

A simple implementation of the classic Minesweeper in your terminal.

Motivation

Just wanted a simple project to practice C.

We have 4 options:
1 - borad size 8x8
2 - borad size 12x12
3 - borad size 15x15
4 - custom size
